# Warehouse Management System (Java)

Учебный проект системы управления складом на Java с графическим интерфейсом (JavaFX).

# О проекте
Курсовая работа по дисциплине "Алгоритмы и структуры данных". Реализация контейнера на основе комбинированной структуры "Двунаправленный список массивов-очередей" для хранения данных склада.

# Основные функции
- Управление структурой склада (секции, ячейки)
- Графический интерфейс на JavaFX
- Сохранение и загрузка данных из файла
- Реализация собственных структур данных:
  - Двунаправленный связный список
  - Статическая кольцевая очередь

# Cтек
- Язык: Java
- GUI: JavaFX, FXML
- Структуры данных: Связные списки, очереди
- Архитектура: ООП, паттерн Singleton
- Сборка: Maven
